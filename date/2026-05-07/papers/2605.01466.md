# SplAttN: Bridging 2D and 3D with Gaussian Soft Splatting and Attention for Point Cloud Completion

**【点云补全的跨模态注意力方法】**

**arXiv**: https://arxiv.org/abs/2605.01466  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2605.01466  
**HF Paper**: https://hf-mirror.com/papers/2605.01466  
**HF Votes**: 2

## 简要摘要

*计算机视觉;三维视觉;点云补全*

现有的点云补全方法在将稀疏点云硬投影到图像平面时会导致跨模态联系中断，即跨模态熵崩溃问题。
提出SplAttN模型，使用可微分高斯软栅格化实现连续稠密的图像平面表示，有效建立跨模态连接，在多个数据集上取得领先性能并保持对视觉线索的鲁棒依赖。

## 深度解析

(1) **论文所在的任务**  
- **任务名称**：点云补全（Point Cloud Completion）  
- **任务输入**：一个不完整、稀疏的 3D 点云（可能附带对应的 2D 图像）  
- **任务输出**：一个完整、稠密的 3D 点云  
- **任务功能**：根据残缺的 3D 形状信息（和可能的 2D 图像信息），推测出完整的 3D 形状结构，常用于自动驾驶、机器人感知、三维重建等领域。  

(2) **想解决的问题**  
现有的多模态点云补全方法虽然效果好，但**理论机制不清晰**。作者发现，传统方法使用“硬投影”将稀疏点云映射到 2D 图像平面，会导致投影结果极为稀疏，使得**视觉信息无法有效传递到 3D 空间**，作者称之为“跨模态熵坍塌”。这导致模型难以真正利用图像信息，反而可能退化成只依赖 3D 模板的单模态模型。

(3) **核心思路**  
不用“硬投影”，改用**可微分高斯泼溅**（Differentiable Gaussian Splatting）来建立 2D 与 3D 之间的桥梁。  
- 把 3D 点云投影到 2D 时，不是直接映射到离散像素，而是用高斯分布表示每个点在图像平面上的“软”密度分布，得到**连续、稠密的 2D 表示**。  
- 这样梯度可以回传，视觉特征更容易与几何特征对齐，从而建立真正的跨模态连接。  

(4) **结果怎么样**  
- 在 PCN、ShapeNet-55/34 等标准数据集上取得了**SOTA**效果。  
- 在真实场景数据集 KITTI 上做了“压力测试”：将图像信息移除后，基线模型性能几乎不变（说明它们其实没用到图像信息），而 SplAttN 性能明显下降，证明它**真正依赖视觉信息**，建立了有效的跨模态连接。  

(5) **模型结构**  
**基模**：没有直接使用某个现有大模型，但借鉴了 Transformer 中的注意力机制（Cross-Attention、Self-Attention）和 PointNet++ / DGCNN 中的局部特征提取模块（EdgeConv）。  

**整体流程**（对照图 1）：  

1. **双分支特征提取**  
   - **GS-Bridge 分支**：  
     - 输入点云通过可微分高斯泼溅投影到 2D 平面，得到稠密的视觉特征 \( \mathcal{F}_{vis} \)。  
     - 几何分支提取点云的全局几何特征 \( \mathcal{F}_{geo} \)，并主动去查询（Cross-Attention）视觉特征，实现跨模态对齐。  
   - **局部编码器分支**：  
     - 使用 EdgeConv 提取局部拓扑特征，再经过多头自注意力进一步提炼，得到局部细节特征 \( \mathcal{F}_l \)。  

2. **全局-局部解码器**  
   - 用全局特征 \( \mathcal{F}_g \) 通过 MLP 预测一个稀疏的初始骨架点云 \( \mathcal{P}_0 \)。  
   - 通过多级上采样逐步增加点数（\( \mathcal{P}_0 \rightarrow \mathcal{P}_2 \)）。  
     - 每一级上采样时，先用**结构自注意力**保持几何一致性，再用**交叉注意力**将局部特征 \( \mathcal{F}_l \) 作为 Key/Value 注入，细化细节。  
   - 最终输出完整的稠密点云。  

**简单来说**：  
输入稀疏点云 → 用高斯泼溅软投影到 2D → 提取全局视觉与几何特征并跨模态对齐 → 同时提取局部几何细节 → 融合全局与局部特征，逐步上采样生成完整点云。
