# End-to-End Autoregressive Image Generation with 1D Semantic Tokenizer

**【端到端自回归图像生成】**

**arXiv**: https://arxiv.org/abs/2605.00503  
**AlphaXiv**: https://www.alphaxiv.org/zh/overview/2605.00503  
**HF Paper**: https://hf-mirror.com/papers/2605.00503  
**HF Votes**: 2

## 简要摘要

*计算机视觉;图像生成;视觉自回归模型*

自回归图像生成依赖视觉分词器将图像压缩为紧凑的隐式表示。
我们设计了一个端到端的训练流程，联合优化重建与生成，并利用视觉基础模型改进一维分词器，取得了先进的生成效果。

## 深度解析

(1) **这篇论文所在的任务是什么？**  
- **任务输入**：一张图像（例如 256×256 像素的图片）。  
- **任务输出**：生成一张新的、高质量的图像（例如风格、内容相似的图片）。  
- **任务功能**：实现**图像生成**，具体来说是通过**自回归（Autoregressive, AR）建模**的方式生成图像，类似于语言模型逐个预测下一个词，但这里是逐个预测图像的下一个“视觉词”（token）。

---

(2) **这篇论文想解决什么问题？**  
传统自回归图像生成通常分为两步：  
1. 先用一个**图像分词器（Tokenizer）** 把图像压缩成离散的 token 序列（类似把图像“编码”成一串数字）。  
2. 再用一个**自回归模型**（如 Transformer）学习预测这些 token 序列。  

**问题在于**：  
- 分词器只负责**重建图像**（尽量还原原图），不关心生成模型能否很好地预测 token。  
- 如果分词器生成的 token 序列不适合自回归预测，生成效果会变差。  
- 此外，很多分词器输出的是**二维（2D）网格状 token**，而自回归模型需要**一维（1D）序列**，这会导致 token 之间的依赖关系不匹配。

---

(3) **论文的核心思路是什么？**  
1. **设计一维分词器（1D Tokenizer）**：  
   - 不再输出 2D 网格 token，而是直接输出一串 1D token 序列，更适合自回归预测。  
2. **端到端联合训练**：  
   - 不再分两步训练，而是**同时训练分词器和自回归模型**，让生成任务的反向传播直接指导分词器学习。  
3. **引入语义对齐**：  
   - 利用视觉基础模型（如 CLIP、DINO）的语义信息，帮助分词器学习更有意义的 token 表示。  
4. **新增“自回归预测重建损失”**：  
   - 在训练时，不仅计算 token 预测损失，还让自回归模型预测的 token 序列**解码回像素空间**，计算重建损失，确保生成的图像质量。

---

(4) **结果怎么样？**  
在 ImageNet 256×256 图像生成任务上：  
- 取得了 **FID 分数 1.48**（越低越好），这是目前**无引导（without guidance）生成中的最好结果**。  
- 相比两阶段训练方法，端到端训练显著提升了生成质量。  
- 模型规模扩大时性能持续提升（见图1c），说明方法具有良好的可扩展性。

---

(5) **模型结构及流程**  
**基模**：基于 **Vision Transformer（ViT）** 和 **Transformer 自回归模型**。  

**整体流程**（对应图1b）：  
1. **输入图像**（如 256×256）进入 **1D 分词器**：  
   - 分词器是一个 ViT 编码器 + 量化模块，把图像编码成一串 1D 离散 token（例如 256 个 token）。  
2. **自回归模型**（Transformer）接收 token 序列：  
   - 按顺序预测下一个 token（类似 GPT 预测下一个词）。  
3. **联合训练**：  
   - 计算 **token 预测损失**（交叉熵）。  
   - 同时，把预测的 token 序列输入**解码器**，重建图像，计算 **像素重建损失**（APR 损失）。  
   - 分词器和自回归模型**一起更新参数**。  
4. **语义对齐**：  
   - 在分词器的中间层，加入视觉基础模型的特征进行对齐（隐式对齐，不破坏 1D 结构）。  
5. **生成时**：  
   - 自回归模型从起始 token 开始，逐步预测 token 序列，再通过解码器生成最终图像。

**简单总结**：  
这篇论文提出了一种**端到端训练的一维语义分词器**，让图像生成的分词和建模过程协同优化，同时引入外部语义知识，最终在 ImageNet 生成任务上取得了领先效果。
